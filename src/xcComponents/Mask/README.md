# XcMask

遮罩层

## Design

基础组件，可以配合各种需要遮罩效果的组件使用，例如自定义模态框、开屏广告等组件。

## Feature

- 遮罩层颜色

## Props

| 参数                | 说明                                                         | 必填 | 类型     | 默认值             |
| ------------------- | ------------------------------------------------------------ | ---- | -------- | ------------------ |
| show                | 展示                                                         |      | boolean  | false              |
| maskBackgroundColor | 设置遮罩层颜色                                               |      | string   | rgba(0, 0, 0, 0.6) |
| onClickMask         | 点击遮罩区域触发的事件，可以在该事件中将show设置为false关闭遮罩层。 |      | Function |                    |
| children            | 子元素                                                       |      |          |                    |
