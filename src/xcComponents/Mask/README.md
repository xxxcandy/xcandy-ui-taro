# XcMask

遮罩层

## Design

基础组件，可以配合各种需要遮罩效果的组件使用。

### Feature

- 遮罩层颜色

## Props

| 参数                | 必填 | 类型     | 默认值             |
| ------------------- | ---- | -------- | ------------------ |
| show                |      | boolean  | false              |
| maskBackgroundColor |      | string   | rgba(0, 0, 0, 0.6) |
| onClickMask         |      | Function |                    |
| children            |      |          |                    |
